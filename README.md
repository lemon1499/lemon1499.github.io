# yingliyu.github.io

:tada: This is a private tech blog for sharing！

# 启动

npm run start

# 新建 tags

hexo new page tags

# 新建 categories

hexo new page categories

# 新建 photos

hexo new page photos

# 新建一篇文章

hexo new [layout] title

# build

hexo generate

# 部署

hexo deploy

This repo uses GitHub Actions to publish a hexo blog into GitHub Pages.
